Explication du plan diagnostique et thérapeutique

Mme Burnstein, je pense que vous avez très probablement ce qu'on appelle une gastro-entérite. Cela signifie que la muqueuse de votre tractus intestinal est devenue enflammée. Lorsqu'elle devient enflammée comme cela, l'intestin peut cesser de faire son travail, qui est de digérer les aliments. Cela peut se produire pour plusieurs raisons, la plus courante étant une infection virale, mais cela peut aussi être causé par des bactéries. La différence entre ces deux est que le virus est un agent infectieux qui ne peut être traité qu'avec des mesures de soutien, tandis qu'une bactérie est un microbe que nous pouvons traiter avec des antibiotiques. C'est pourquoi il est important de connaître la différence, donc je vais prescrire plusieurs études pour essayer de déterminer ce qui se passe avec vous aujourd'hui. Ces études comprennent une analyse de sang et aussi un échantillon de selles de votre part pour analyse. Maintenant, la gastro-entérite n'est pas la seule possibilité - juste la plus probable. Une autre possibilité est que vous pourriez être enceinte, mais cela n'explique pas tous vos symptômes ; ainsi, même si vous êtes enceinte, je pense qu'il y a une autre cause probable. Cependant, comme vous avez eu des rapports sexuels non protégés récemment, j'aimerais également tester cela. Une autre possibilité est que votre vésicule biliaire ait des calculs qui bloquent le canal qui s'étend d'elle à l'intestin. Ces calculs sont assez courants, et cela pourrait expliquer certains symptômes que vous avez. Pour enquêter davantage, je voudrais obtenir une échographie de votre abdomen. Cela implique de placer une petite sonde sur votre abdomen et de regarder un écran à proximité. Cela ne devrait pas faire mal mais peut être légèrement inconfortable. Permettez-moi de sortir pour organiser ces tests avec mon infirmière afin que nous puissions commencer. Y a-t-il un autre aspect de vos soins de santé dont nous n'avons pas déjà discuté ? D'accord alors, merci.

Rappels théoriques - Nausées/Vomissements et Cholécystite

Signe de Murphy

Test clinique pour cholécystite aiguë. Technique : main posée sous rebord costal droit, demander inspiration profonde. Positif si arrêt inspiratoire par douleur lors palpation vésicule. Spécificité 79-96%, sensibilité 65%. Toujours comparer avec côté gauche (doit être indolore).

Diagnostic différentiel nausées/vomissements aigus

Digestifs : gastro-entérite, cholécystite, pancréatite, appendicite, occlusion. Gynéco : grossesse, GEU, torsion ovarienne. Métaboliques : acidocétose diabétique, urémie, hypercalcémie. Neurologiques : migraine, HIC, labyrinthite. Médicamenteux : opioïdes, digoxine, chimiothérapie.

Évaluation déshydratation adulte

Légère (<5%) : soif, bouche sèche. Modérée (5-10%) : pli cutané, tachycardie, hypotension orthostatique, oligurie. Sévère (>10%) : confusion, extrémités froides, anurie, choc. Perte poids = meilleur indicateur si connu.

Rappels thérapeutiques
  • Réhydratation : privilégier IV si vomissements (NaCl 0.9% ou Ringer)
  • Antiémétiques : ondansétron 4-8mg IV première ligne, métoclopramide si échec
  • Cholécystite : antibiotiques (céfazoline + métronidazole), chirurgie dans 72h
  • Grossesse : adapter traitement (éviter métoclopramide T1, préférer doxylamine)
  • Gastro-entérite : réhydratation, régime progressif, antibiotiques rarement
  • Surveillance : diurèse, ionogramme, signes péritonéaux
Examens complémentaires
  • β-HCG : systématique femme en âge procréer avec nausées/vomissements
  • NFS : hyperleucocytose si infection bactérienne ou cholécystite
  • Ionogramme : évaluer déshydratation, troubles électrolytiques
  • Bilan hépatique : élévation si cholécystite (PAL, bilirubine)
  • Lipase : × 3 normale si pancréatite (plus spécifique que amylase)
  • US abdominale : épaississement paroi vésiculaire >4mm, liquide péri-vésiculaire, Murphy échographique
  • Analyse urine : densité élevée si déshydratation, corps cétoniques si jeûne
